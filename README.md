# Manipulation Code Introdution

# Manipulation Code Structure
The manipulation code is organized into several subdirectories, each containing specific components of the manipulation system.
Main mission code is about Drake manipulation, which includes the main code for the manipulation tasks, such as the control algorithms, state estimation, and trajectory planning.
